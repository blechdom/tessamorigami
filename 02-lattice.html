THE LATTICE SCANNER LINE = PLAYHEAD

HOW IT READS

The line is
the playhead.

There is no arbitrary row of moving dots. The line is the only clock. At each instant, the visible dots are the unique intersections between that line and every visible repeated tile boundary. Shared edges and shared vertices are deduplicated before drawing or sounding.

Height maps logarithmically to pitch and horizontal position maps to stereo. Edge direction changes tone color. When there are more contacts than the browser should voice clearly, the display still shows every dot while the sound uses an evenly spaced, deterministic subset. The voice cap makes that choice visible and adjustable.

FIELD NOTES / 02

A lattice is one object.

The tiles come from the genuine tactile-js isohedral engine. Curvature changes matching edge pairs, with zero producing exactly straight boundaries. Density changes tile scale; extent changes how much of the field participates.

The scanner intersects the actual sampled boundaries rather than placing decorative playheads. Coincident contacts are merged in screen space. If contacts exceed the selected polyphony, evenly distributed points along the line are chosen in a stable order so the sound does not flicker between arbitrary voices.

Drag the canvas to place the scan line. Use the arrow keys for fine offset and angle changes. Sound is synthesized live and begins only after pressing the transport.
